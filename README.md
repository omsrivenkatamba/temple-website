# Bhagavathi Sri Venkatamba Temple Website

Official website for Bhagavathi Sri Venkatamba Temple, Karaballavolu Village, Andhra Pradesh.

## Live Website
https://omsrivenkatamba.github.io/temple-website/

## Features
- Responsive design for all devices (mobile, tablet, desktop)
- Photo gallery with auto-sliding hero section
- Team member display from community-team folder
- Donation system with UPI QR code and bank details
- Contact form for inquiries
- Information about temple activities and services
- Social media integration
